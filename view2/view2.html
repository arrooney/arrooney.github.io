AlbertaSat is a student group at the University of Alberta, we are a team of undergraduate students and faculty advisors who build satellites!
Ex-alta 1 was not only AlbertaSat's first satellite, but also Alberta's first satellite. Ex-alta 1 has since left it's orbit, and lives on in memory. We are now working on Ex-alta 2, an earth imaging satellite that will be designed to detect, and predict forest fires using a multispectral imager, and phenology.
My role on the team is as the Software Lead, in this role I have spent the summer with an NSERC USRA grant reasearching and implementing an open source network stack for use on Ex-alta 2, as well as assisting in developing and integrating an Inter-integrated Circuit driver (I2C) for the data bus. Over the summer I was also tasked with designing and implementing automation for the Ex-alta 1 ground station. This was because operators had to be awake at all hours to communicate with the spacecraft when overhead. I was able to create numerous bash, and FORTH scripts, with hooks to C code that automated many routine telemtry collection tasks.
Another small project of mine was my path finding robot. My friend and I created a robot that did two things: first, an image of a maze was input into the A* path finding algorithm, written in python. Next, the directions for how to navigate the maze was output to a robot with stepper motors over bluetooth; the Arduino controlled and C++ programmed robot then found it's way through the maze!
Having had some experience with low-level communication protocols, my friend and I were able to implement laser communication with hamming error correction code and packet acknowledgement. The setup included two Arduino nanos, each with a laser and a photoresistor. The Arduinos were fully capable of sending messages to one another with the ability to retry until each packet is acknowledged, and correct bit-flips!
