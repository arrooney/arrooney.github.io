About Me

I have project and classroom experience with software design & development, project management, VHDL logic design, custom floorplanning with Cadence, and have experience fabricating multilayer nanoscale electronic and MEMS systems at UAlberta's NanoFAB facility.

AlbertaSat

AlbertaSat is a University of Alberta vehicle project. Our team is creating our second satellite called "Ex-Alta 2"; its mission is to collect multispectral image data to assist scientists in making assesments and predictions about forest fires in Alberta.

I joined the project in my first year, and was assigned to the software team. I spent the summer of my first year working under Prof. Duncan Elliott with NSERC funding to develop hardware interfaces for the network stack being used on the space craft, and also to develop automation capabilities for the "Ex-Alta 1" satellite's ground station. In this time, I learned a lot about the project, and how to approach the challenges of a large project.

I became the lead of the software team in my second year, and have put a great deal of effort into improving my ability to communicate and delegate tasks to distribute the workload of developing the open-source embedded software. My goal for this project is to develop high-quality code that can be useful for other low-buget space programs. To do this we work to ensure we have smooth development operations to hopefully attract outside contributors, and create non-hardware-specific designs so that the software can be run on any platform.

Also on this project, I have had the opportunity to go to high schools around the province to take part in the education initiatives that make AlbertaSat unique. In these education outreach sessions I have developed programs to introdce aspects of the C programming language to high school students.

All of the opern-cource code for the Ex-Alta 2 satellite is written in C, and uses the FreeRTOS embedded operating system and the "Cubesat Space Protocol" network stack to implement a service-oriented arcitecture on the space craft. All the on-board-computer code can be found here!

  • Technologies used: C, Cortex R4F microcontroller, FreeRTOS, "CSP" Network Stack, Git (extensively), hardware protocols (I2C, CAN, serial), and technical project management

SEDS High Altitude Balloon Mission

In August 2018 me and a team of about 6 of my peers constructed a mock-up of the Ex-Alta 2 satellite payload, which was a multi-spectral imager. We designed the electronics, mechanical, and software for the experiment to be able to fly on a high-altitude balloon. We launched the experiment on the balloon (pictured) which took the camera to the stratosphere. I did the sofrware designs for the Raspberry PI to interface with the GigE IP camera which operated via ethernet. The Raspberry PI also operated an IMU to track the accelleration and temperature of the experiment.

  • Technologies used: Raspberry PI, GigE IP Camera, Image Processing, lots of teamwork

Ex-Alta 2 Ground Station

The Ex-Alta 2 ground station is being created from scratch. After working extensively to improve the Ex-Alta 1 ground station, I decided we needed a change due to the obscure scripting language used (FORTH).

The Ex-Alta 2 ground station uses Python with binding to the open source network stack "Cubesat Space Protocol" to server as either a terminal interface, or as a back-end for the web-interface to the satellite in-orbit using GNUradio with a Software Defined Radio to communicate over UHF Ameteur frequencies.

This ground station can also be run in a docker image to decrease dependencies on the host environment.

See the code

  • Technologies used: Python, "CSP" network stack, C, GNUradio

A* Path Finding Robot

Another small project of mine was my path finding robot. My friend and I created a robot that did two things: first, an image of a maze was input into the A* path finding algorithm, written in python. Next, the directions for how to navigate the maze was output to a robot with stepper motors over bluetooth; the Arduino controlled and C++ programmed robot then found it's way through the maze!
